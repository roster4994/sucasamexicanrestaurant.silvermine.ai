# Photo Collage

Upload your photos here for the home page collage section.

## Recommended:
- Image formats: JPG, PNG, WebP
- Recommended size: 800x600 pixels or larger
- Aspect ratio: 4:3 or 16:9 works best
- Add 4-8 photos for a balanced collage

## Usage:
After uploading photos, update the `collageImages` array in `src/pages/index.astro` with the filenames.
