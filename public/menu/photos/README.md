# Menu Photos

Upload your menu item photos to this folder.

## Recommended Guidelines
- Use JPG or PNG format
- Recommended size: 800x600 pixels or larger
- Use descriptive filenames (e.g., `tacos-al-pastor.jpg`, `enchiladas-verdes.jpg`)

Once uploaded, photos can be referenced in the menu page at `/menu/photos/your-image-name.jpg`
